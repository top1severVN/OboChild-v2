# 🚀 OboChild Framework
**A powerful, one-liner style library for Science, String processing, and Web.**
*Một thư viện mạnh mẽ, phong cách "xử lý trong một dòng" dành cho Khoa học, Xử lý chuỗi và Web.*

## ✨ Features / Tính năng:
- **stdio:** is the standard library for input and output.
- **science:** Physics formulas (Work, Power, Potential Energy).
- **weblib:** Fetching data and JSON from URLs easily.
- **restr:** Advanced string manipulation and Regex support.
- **omath:** It's a specialized mathematics library; it contains mathematical functions and constants from around the world. It includes sequences like Fibonacci, geometric constants, and geometric formulas.
- **stdrand:** It's a library specializing in random numbers, with many different types of random numbers.
- **loop:** This is a library specializing in forever and custom loop types, including ring.
- **condilib:** is a library that supports conditions
- **press:** It is a library that supports button recognition.
- **stdvars:** This is a library that helps with variable management; it's a bit lengthy, so please bear with me. I recommend using Python's standard variable assignment method.
